# Исправление проблемы с сетью Docker

## Проблема
Docker не может подключиться к npm registry для установки зависимостей.

## Решения

### 1. Проверьте интернет-соединение
Убедитесь, что у вас есть стабильное интернет-соединение.

### 2. Отключите прокси в Docker Desktop (если не нужен)
1. Откройте Docker Desktop
2. Settings → Resources → Proxies
3. Отключите прокси или настройте правильно

### 3. Увеличьте таймаут npm (уже сделано в Dockerfile)
Таймаут уже увеличен до 300 секунд.

### 4. Попробуйте собрать образ в другое время
Иногда npm registry может быть перегружен.

### 5. Используйте локальные зависимости (если есть)
Если у вас уже установлены node_modules локально, можно скопировать их в образ.

### 6. Проверьте файрвол
Убедитесь, что файрвол не блокирует Docker.
